# AI-Powered E-Commerce Analytics System

## Project Overview
This project is an end-to-end E-Commerce Analytics System built using Python, SQL, Machine Learning, and Power BI.
The project analyzes e-commerce transaction data to generate business insights, customer segmentation, and interactive dashboards.

---

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- SQL Business Queries
- Customer Segmentation using K-Means Clustering
- Interactive Power BI Dashboard

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- MySQL
- Power BI

---

## AI Integration
Implemented customer segmentation using:
- RFM Analysis
- K-Means Clustering

---

## Business Insights
- Revenue trends
- Customer behavior analysis
- Top-selling products
- High-value customer identification
- Peak shopping hours
